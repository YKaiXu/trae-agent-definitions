# TRAE智能体定义

本项目包含11个专业智能体，可一键导入TRAE IDE使用。

## 智能体列表

| 智能体 | ID | 触发关键词 |
|--------|-----|-----------|
| Frontend Architect | frontend-architect | 前端架构、技术选型、组件设计、性能优化 |
| Backend Architect | backend-architect | 后端架构、API设计、数据库、微服务 |
| UI Designer | ui-designer | UI设计、组件库、响应式、设计系统 |
| API Test Pro | api-test-pro | API测试、性能测试、安全测试 |
| E2E Test Engineer | e2e-test-engineer | E2E测试、用户流程、跨浏览器 |
| AI Integration Engineer | ai-integration-engineer | GPT集成、RAG知识库、AI Agent |
| DevOps Architect | devops-architect | CI/CD、K8s、监控、GitOps |
| Performance Expert | performance-expert | 性能优化、压力测试、瓶颈诊断 |
| Data Analyst | data-analyst | 数据分析、可视化、SQL查询 |
| Compliance Checker | compliance-checker | 合规审查、GDPR、隐私政策 |
| Product Manager | product-manager | 需求分析、产品设计、竞品分析 |

## 每个智能体包含

1. **提示词** - 角色定义和核心能力
2. **何时调用** - 必须调用场景、关键词触发、调用优先级

## 使用方法

在TRAE IDE中创建智能体时，直接复制对应md文件的内容即可。
