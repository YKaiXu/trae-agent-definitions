# TRAE智能体定义

本项目包含11个专业智能体的完整定义，用于TRAE IDE一键导入。

## 智能体列表

| 智能体 | ID | 描述 |
|--------|-----|------|
| Frontend Architect | frontend-architect | 前端架构设计、组件开发、性能优化 |
| Backend Architect | backend-architect | 后端架构、API设计、数据库优化 |
| UI Designer | ui-designer | UI/UX设计、组件库、设计系统 |
| API Test Pro | api-test-pro | API功能测试、性能测试、安全测试 |
| E2E Test Engineer | e2e-test-engineer | 端到端测试、用户流程测试 |
| AI Integration Engineer | ai-integration-engineer | AI集成、RAG知识库、智能功能 |
| DevOps Architect | devops-architect | CI/CD、云基础设施、容器化 |
| Performance Expert | performance-expert | 性能分析、优化、压力测试 |
| Data Analyst | data-analyst | 数据分析、可视化、业务洞察 |
| Compliance Checker | compliance-checker | 合规审查、法律风险评估 |
| Product Manager | product-manager | 需求分析、产品设计、产品规划 |

## 使用方法

每个智能体定义在 `definitions/` 目录下，可根据TRAE IDE的导入功能一键导入使用。
